## 홈 화면 — '다가오는 시험' → '지금 접수 가능'

### 변경 배경
"다가오는 시험"은 이미 접수가 마감된 시험도 표시되는 문제가 있었음.  
사용자가 **지금 당장 접수할 수 있는 시험**을 바로 확인할 수 있도록 변경.

### 로직
- `register_start ≤ 오늘 ≤ register_end` 조건을 만족하는 시험만 표시
- 카드에 **접수 마감일 + D-day**, **시험일** 함께 노출
- 마감 **3일 이내** → 접수 마감일 분홍색 강조
- 접수 중인 시험 없을 때 → "다음 접수 → OO · mm.dd 시작" 안내

### 변경 파일
- `src/pages/Home/index.jsx`

## 버그 수정 — 홈 자격증 클릭 시 항상 ADsP 정보로 이동

### 원인
홈 화면 자격증 버튼이 `/info?exam=sqld` 형태로 링크했으나,  
Info 페이지가 URL 파라미터를 무시하고 `useState('adsp')` 초기값만 사용.

### 수정
`useSearchParams`로 `?exam=` 파라미터를 읽어 카테고리 탭과 선택 자격증을 동시에 초기화.

```jsx
const paramExam = EXAMS.find((e) => e.id === searchParams.get('exam'));
const [catFilter, setCatFilter] = useState(paramExam?.category ?? 'it-data');
const [selected, setSelected] = useState(paramExam?.id ?? 'adsp');
```

## 컴활·OPIc 상시시험 표시 추가

### 변경 배경
컴퓨터활용능력 1급과 OPIc은 정해진 접수 기간 없이 상시 응시 가능.  
홈 화면 "지금 접수 가능" 섹션에서 날짜 기반 카드와 별도로 항상 표시되어야 함.

### 변경 내용
- EXAMS에 `ongoing: true` 플래그 추가 (컴활, OPIc)
- 홈 화면: 접수 중인 시험 카드 아래 상시시험 카드 표시 (초록 "상시" 배지)
- 상시시험 카드 클릭 시 해당 시험 정보 페이지(`/info?exam=`)로 이동
- Info 페이지 시험 타이틀에 "상시" 배지 표시

### 변경 파일
- `src/data/exams.js` — `ongoing: true` 추가 (computer, opic)
- `src/pages/Home/index.jsx` — 상시시험 카드 렌더링 추가
- `src/pages/Info/index.jsx` — 상시 배지 표시 추가
