## 로그인 전 시험 → 로그인 후 자동 업로드

### 문제

기존 코드는 로그인 시 DB 데이터로 로컬 목록을 덮어써버려,  
비로그인 상태에서 추가한 시험이 로그인 후 사라지는 문제가 있었다.

### 해결 — `src/hooks/useMyExams.js`

```
로그인 감지 (user?.id 변경)
  ↓
로컬 목록 캡처 (getLocalExams)
  ↓
DB fetch
  ↓
toUpload = 로컬에만 있고 DB에 없는 항목
  ↓
toUpload > 0 → upsert (onConflict: 'user_id,exam_id')
  ↓
merged = Set(dbIds + toUpload) → setState + localStorage 갱신
```

- `upsert({ onConflict: 'user_id,exam_id' })`: 이미 DB에 있는 항목은 오류 없이 무시
- `uploadedCount` 반환 → My 탭에서 토스트 표시에 활용

### 업로드 완료 토스트 — `src/pages/My/index.jsx`

- `uploadedCount > 0` 시 초록색 토스트 3초 표시
- "기기에 저장된 시험 N개를 클라우드에 업로드했어요"
