## 캘린더 날짜 상세 패널 — `src/pages/Schedule/index.jsx`

### 동작

Apple Calendar 스타일 참고.  
날짜를 탭하면 캘린더 하단에 해당 날의 일정 목록이 표시됨.

```
[캘린더 그리드]
  ↓ 날짜 탭
┌─────────────────────────────────┐
│ 6월 22일 (월)              ✕   │
├─────────────────────────────────┤
│ ▌ ADsP            IT·Data  D-3  │
│ ▌ SQLD            IT·Data  D-3  │
└─────────────────────────────────┘
```

### 구현 내용

- `CalendarView`에 `selectedDay` 상태 추가
- 날짜 셀 클릭 → `selectedDay` 토글 (같은 날 재탭 시 닫힘)
- 선택된 날짜: 보라색 배경(`#EDE8FF`) + 날짜 숫자 보라색·볼드
- 월 이동 시 `useEffect`로 선택 자동 해제
- 패널 내용: 시험명 / 일정 유형 / 카테고리 뱃지 / D-Day 뱃지
- 일정 없는 날: "일정이 없는 날이에요" 안내

### catFilter 연동

`CalendarView`에 `catFilter` prop 추가.  
카테고리 필터 적용 상태가 캘린더 점(dot) 표시와 상세 패널 모두에 반영됨.

### 변경 파일
- `src/pages/Schedule/index.jsx`
  - `useEffect` import 추가
  - `CalendarView` props: `year, month` → `year, month, catFilter`
  - `selectedDay` 상태 및 상세 패널 UI 추가
  - 3개 이상 일정 있는 날: 회색 점 추가로 truncation 표시
