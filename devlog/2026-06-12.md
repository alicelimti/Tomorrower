# 2026-06-12 — Vercel 404 수정 & v2.0 전면 개편 & 테마 컬러 변경

## 오전 — 버그 수정: SPA 새로고침 404

**증상**  
`/schedule`, `/quiz` 등 하위 경로에서 새로고침 시 `404: NOT_FOUND` 발생

**원인**  
React Router 클라이언트 사이드 라우팅 특성상 해당 경로에 실제 파일이 없음.  
새로고침 시 Vercel 서버가 파일을 찾다가 404 반환.

**해결**  
`vercel.json` 추가 — 모든 경로를 `index.html`로 rewrite.

```json
{ "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }] }
```

## 야간 — 실제 시험 일정 데이터 업데이트

### 배경
기존 `EXAM_SCHEDULE`은 예시 데이터로 채워져 있었음.  
한국데이터산업진흥원 공식 홈페이지 이미지를 기반으로 실제 2026년 일정으로 전면 교체.

### 업데이트 내용

| 시험 | 회차 | 시험일 | 접수 기간 | 합격 발표 |
|---|---|---|---|---|
| ADSP | 제49회 합격발표 | — | — | 6.5 |
| ADSP | 제50회 | 8.8 (토) | 7.6~7.10 | 8.28 |
| ADSP | 제51회 | 10.31 (토) | 9.28~10.2 | 11.20 |
| ADP | 제37회 필기 | 8.8 (토) | 7.6~7.10 | 9.4 |
| ADP | 제37회 실기 | 10.17 (토) | 9.14~9.18 | 11.13 |
| SQLD | 제61회 합격발표 | — | — | 6.19 |
| SQLD | 제62회 | 8.22 (토) | 7.20~7.24 | 9.11 |
| SQLD | 제63회 | 11.14 (토) | 10.12~10.16 | 12.4 |
| SQLP | 제55회 | 8.22 (토) | 7.20~7.24 | 9.18 |

### 데이터 설계 결정 사항

- **과거 회차 제외**: 타임라인 카드가 `find()`로 첫 번째 매칭 항목을 사용하므로, 과거 회차 데이터를 포함하면 그레이 처리된 지난 일정이 노출됨 → 현재~미래 회차만 포함
- **결과 발표일 단독 포함**: SQLD 제61회는 시험(5.31)은 완료됐으나 합격 발표(6.19)가 미발표 상태 → result 타입 항목만 추가하여 6월 캘린더/목록 뷰에 노출
- **next_exam 연결**: 각 시험의 마지막 회차 카드에서 다음 회차 시험일을 `next_exam` 타입으로 연결하여 타임라인 4단계 완성
