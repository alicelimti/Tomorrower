# 2026-06-12 — Vercel 새로고침 404 에러 수정

## 오늘 한 일

### 버그 수정: SPA 새로고침 시 404: NOT_FOUND

**증상**
- `/schedule`, `/quiz` 등 하위 경로에서 새로고침 시 Vercel 404 에러 발생
- `404: NOT_FOUND / Code: NOT_FOUND`

**원인**
React Router는 클라이언트 사이드 라우팅 방식으로, `/schedule` 같은 경로에 실제 파일이 존재하지 않음.
새로고침 시 Vercel 서버가 해당 경로의 실제 파일을 찾다가 없으면 404를 반환.

**해결**
`vercel.json` 추가 — 모든 경로 요청을 `index.html`로 rewrite하여 React Router가 클라이언트에서 처리하도록 설정.

```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```

**결과**
- 모든 탭에서 새로고침 정상 동작 확인
- 커밋: `fix: Vercel SPA 새로고침 404 에러 수정`
