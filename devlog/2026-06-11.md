### 3. 프로젝트 구조 세팅

```
src/
├── components/layout/BottomNav.jsx   하단 탭바 (5탭)
├── data/
│   ├── exams.js       자격증 6종 데이터 + 시험 일정
│   └── quizBank.js    문제은행 예시 (ADSP 2문제, SQLD 3문제)
├── firebase/config.js Firebase 초기화 (환경변수 연동 준비)
└── pages/
    ├── Home/          홈 — D-Day 카드, 오늘의 퀴즈 배너, 자격증 바로가기
    ├── Schedule/      일정 — 월별 캘린더 + 리스트 뷰, 일정 유형 컬러 코드
    ├── Quiz/          퀴즈 — 시험 선택 → 10문제 → 결과/오답 해설
    ├── Info/          정보 — 합격 컷트라인 테이블, 공식 사이트 링크
    └── My/            마이 — 학습 통계, 주간 그래프, 달성 배지
```

### 5. 구현된 기능

#### 홈
- 오늘 날짜 + 인사 헤더
- 오늘의 퀴즈 배너 (퀴즈 페이지 링크)
- 다가오는 시험 D-Day 카드 (자동 계산)
- 지원 자격증 6종 바로가기 그리드

#### 일정
- 월별 캘린더 뷰 (오늘 날짜 강조, 일정 도트 표시)
- 목록 뷰 전환 (캘린더 ↔ 리스트)
- 월 이전/다음 네비게이션
- 일정 유형별 컬러 코드 범례 (접수시작/접수마감/시험일/합격발표)

#### 퀴즈
- 시험별 or 전체 랜덤 선택
- 4지선다 문제 풀기
- 정답/오답 즉시 피드백 + 해설
- 완료 후 점수 및 오답 리뷰

#### 정보
- 자격증별 탭 전환
- 합격 기준 / 과락 기준 / 평균 합격률 카드
- 시험 과목 목록
- 공식 사이트 외부 링크
- 전체 자격증 합격 컷트라인 비교 테이블

#### 마이
- 프로필 카드 (Firebase 연동 전 placeholder)
- 이번 주 학습 막대 그래프
- 총 풀이 / 정답률 / 연속일 통계 카드
- 달성 배지 6종 (첫 퀴즈, 3일 연속, 7일 연속, 만점왕, 원서 접수, 합격)
- 스마트 알림 토글 UI
