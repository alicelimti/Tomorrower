## 소개

자격증 시험 일정 관리, 퀴즈 풀기, 내 시험 D-Day 추적을 한 곳에서.  
PWA로 제작되어 모바일 홈 화면에 설치해 앱처럼 사용할 수 있습니다.

### 주요 기능

- 📅 **시험 일정** — 타임라인·캘린더·목록 3가지 뷰, 날짜 탭 시 상세 일정 패널
- 🎯 **My 시험** — 응시할 시험 추가/삭제, D-Day 카운트다운
- 🔐 **Google 로그인** — 시험 목록 클라우드 동기화 (기기 간 공유)
- 📝 **퀴즈** — 자격증별 예상 문제 풀기
- 📲 **PWA** — 오프라인 캐싱, 홈 화면 설치, 업데이트 알림
